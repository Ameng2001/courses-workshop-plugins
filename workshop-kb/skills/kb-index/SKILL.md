# Knowledge Base Index

Scan all documents in `studio/kb/` and build a structured index file (`studio/kb/index.yaml`) from their YAML frontmatter metadata.

## Pre-check

1. Verify `studio/kb/` exists. If not, tell the user to run `/workshop-core:init` first.
2. Glob `studio/kb/**/*.md` to find all imported documents
3. If no documents found, report: "知识库为空。请先使用 `/workshop-kb:kb-import` 导入文档。"

## Step 1: Scan Documents

1. Glob `studio/kb/**/*.md` — collect all markdown files across all subdirectories
2. Skip `index.yaml` and any non-document files
3. For each file, read the YAML frontmatter

## Step 2: Parse Metadata

For each document, extract frontmatter fields into a structured record:

```yaml
- file: "relative/path/to/document.md"
  title: "文档标题"
  category: "textbook"
  themes: ["春天", "植物"]
  age_groups: ["prek-4", "k"]
  domains: ["科学", "艺术"]
  methodology: "five-step"
  term: "2025-春季"
  tags: ["花", "观察", "户外"]
  import_date: "2026-03-28"
```

If a document has no frontmatter or invalid frontmatter, flag it for the user and include it with minimal metadata (file path + category from directory).

## Step 3: Build Index

Assemble the index structure:

```yaml
# studio/kb/index.yaml
# Auto-generated by /workshop-kb:kb-index
# Last updated: {current_date}

stats:
  total_documents: {count}
  by_category:
    textbooks: {count}
    philosophy: {count}
    lesson-plans: {count}
    research-records: {count}
    calendars: {count}
  by_age_group:
    prek-3: {count}
    prek-4: {count}
    k: {count}
  by_domain:
    健康: {count}
    语言: {count}
    社会: {count}
    科学: {count}
    艺术: {count}
  themes: ["主题1", "主题2", ...]  # 所有出现过的主题（去重）

documents:
  - file: "..."
    title: "..."
    category: "..."
    themes: [...]
    age_groups: [...]
    domains: [...]
    methodology: "..."
    term: "..."
    tags: [...]
    import_date: "..."
  - ...
```

## Step 4: Write Index

Write the assembled index to `studio/kb/index.yaml`.

## Step 5: Report

Display indexing results:

```
✅ 知识库索引已更新

📊 统计:
- 文档总数: {total}
- 区编教材: {textbooks} | 园本理念: {philosophy}
- 历年教案: {lesson-plans} | 教研记录: {research-records}
- 学期日历: {calendars}

📌 覆盖主题: {theme_list}
📌 覆盖年龄段: {age_groups}

⚠️ 问题文档: {issues_count}
{list of documents with missing/invalid frontmatter}
```

## Out of Scope

- This skill does NOT import documents (use `kb-import`)
- This skill does NOT perform full-text search (use `kb-query`)
- The index is metadata-based, not full-text — it enables tag-based filtering
