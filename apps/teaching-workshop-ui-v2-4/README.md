# 教研工坊 UI Prototype V2.4

这版聚焦 Claude 风格的工作方式：

- 左侧：`Project / Thread`
- 中间：`rich-rendered conversation surface`
- 右侧：`Course Context Rail`

包含 7 个关键状态：

- `index.html`：Empty State
- `starter-selected.html`：starter 选中态
- `create-project.html`：Create project 入口
- `create-project-scratch.html`：从零开始
- `create-project-import.html`：导入已有课程资料
- `create-project-existing.html`：连接现有课程工作区
- `studio-thread.html`：正式工作态
