# 教研工坊页面原型

本目录包含“教研工坊”前端的静态多页面原型，用于把设计文档落成可浏览页面。

- [project-workspace.html](/Users/liuyameng/.codex/worktrees/8a4e/courses-workshop-plugins/apps/teaching-workshop-ui/project-workspace.html)
- [theme-framing.html](/Users/liuyameng/.codex/worktrees/8a4e/courses-workshop-plugins/apps/teaching-workshop-ui/theme-framing.html)
- [month-matrix.html](/Users/liuyameng/.codex/worktrees/8a4e/courses-workshop-plugins/apps/teaching-workshop-ui/month-matrix.html)
- [week-arrangement.html](/Users/liuyameng/.codex/worktrees/8a4e/courses-workshop-plugins/apps/teaching-workshop-ui/week-arrangement.html)
- [activity-editor.html](/Users/liuyameng/.codex/worktrees/8a4e/courses-workshop-plugins/apps/teaching-workshop-ui/activity-editor.html)
- [hil-gate.html](/Users/liuyameng/.codex/worktrees/8a4e/courses-workshop-plugins/apps/teaching-workshop-ui/hil-gate.html)
- [export-bundle.html](/Users/liuyameng/.codex/worktrees/8a4e/courses-workshop-plugins/apps/teaching-workshop-ui/export-bundle.html)

使用方式：

1. 直接在浏览器中打开任意 `.html`
2. 建议从 `index.html` 开始浏览
3. 页面共享一套 shell、颜色和 Copilot 交互风格

当前原型重点：

- Project Workspace 作为核心，不展示文件树
- Copilot 在不同页面提供不同动作
- HIL 显式呈现
- Matrix 与 Arrangement 两页重点突出结构编排能力
