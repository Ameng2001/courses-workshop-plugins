# Resource Categories / 资源分类规则

本文档定义了 Workshop 课程资源的四大分类体系及分类决策规则。所有课程活动涉及的材料均须归入以下类别之一。

---

## Four Resource Categories / 四大资源类别

### 1. PBL Box / 项目盒

**定义**: 由供应商统一采购、打包配送至各班级的主题材料套装。

**特征**:
- 标准化配置，同年龄段各班内容一致
- 需提前 2–4 周订购
- 由 PBL Box 供应商集中配送

**典型材料**:
- 主题专用工作表 (如水果搭配表、利润计算工作表)
- 手工模板 (如水果模型纸、果汁杯纸)
- 科学工具 (如手动榨汁器、温度计、量杯)
- 主题装饰材料 (如价格标签、果汁优惠券)

**适用场景**: 需要集中采购的材料、主题专属材料、非通用消耗品。

---

### 2. My Journal / 探索足迹袋

**定义**: 幼儿个人的探索记录册，包含工作表、记录页、观察表等，伴随项目全程积累。

**特征**:
- 每人一份，标注姓名
- 随项目推进逐步积累内容
- 项目结束后带回家，作为学习成果展示

**典型材料**:
- 调查问卷 (如果汁调查问卷)
- 观察记录表 (如水果观察记录)
- 绘画/创作页 (如"我设计的果汁"绘画页)
- 计算/思考工作表 (如利润计算工作表)
- 个人反思页

**适用场景**: 凡是幼儿需要亲自填写、绘画、粘贴或收集的个人材料。

---

### 3. Teacher's Supplies / 自备材料

**定义**: 由教师从班级现有资源或本地渠道自行准备的材料。

**特征**:
- 常见易得，无需专门订购
- 各班可能略有差异
- 包含生鲜、易耗品等不适合远程配送的物品

**典型材料**:
- 美术耗材 (颜料、剪刀、胶水、胶带)
- 实物食材 (新鲜水果、冰块)
- 教室家具调整 (桌椅重新摆放为"店铺"布局)
- 清洁用品 (抹布、垃圾袋)
- 基础文具 (白纸、马克笔、铅笔)

**适用场景**: 日常消耗品、生鲜/易腐物品、教室内已有的通用物资。

---

### 4. Media Supplies / 多媒体

**定义**: 数字内容资源——视频、歌曲、图片、应用程序等。

**特征**:
- 可重复使用，无物理采购成本
- 需要播放设备 (投影仪、平板、音箱等)
- 可通过二维码、链接等方式分发

**典型材料**:
- 教学视频 (如 12 Healthy Smoothies 视频)
- 歌曲/音频 (如 Fruit Juice Song)
- 图片参考素材
- 数字绘本/故事
- 互动应用程序

**适用场景**: 演示内容、背景音乐、参考图像、数字互动活动。

---

## Classification Decision Tree / 分类决策树

```
该资源是否为数字内容 (视频/歌曲/图片/应用)?
  → 是 → Media Supplies / 多媒体
  → 否 ↓
该资源是否由幼儿个人保管、填写或记录?
  → 是 → My Journal / 探索足迹袋
  → 否 ↓
该资源是否需要向供应商订购 / 是否为主题专属材料?
  → 是 → PBL Box / 项目盒
  → 否 → Teacher's Supplies / 自备材料
```

---

## Specificity Rules / 材料描述规范

所有资源条目必须包含：**物品名称 + 数量**。

### 反面示例

- "水果" — 缺少具体种类和数量
- "材料" — 完全无法执行
- "工具" — 不知道是什么工具、多少个

### 正面示例

- "苹果 × 5 个"
- "彩色皱纹纸 × 1 包 (20 张)"
- "手动榨汁器 × 2 台"
- "果汁调查问卷 × 25 张 (每人 1 张)"

### 数量标注规则

| 场景 | 标注方式 | 示例 |
|------|---------|------|
| 每人一份 | 总量 = 班级人数 (默认 25) | "水果搭配表 × 25 张" |
| 小组共用 | 注明"每组 1 套" | "手动榨汁器 × 5 台 (每组 1 台)" |
| 全班共用 | 注明"全班共用 1 套" | "利润计算海报 × 1 张 (全班共用)" |
| 消耗品/余量 | 标注合理余量 | "塑料杯 × 50 个 (含备用)" |

> **默认班级规模**: 20–25 名儿童，分 5 组，每组 4–5 人。
