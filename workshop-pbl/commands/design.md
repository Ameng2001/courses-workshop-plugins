Run the full PBL course design pipeline for the specified monthly theme. This chains all 5 design skills in sequence, pausing for user validation at each step.

Pass the monthly theme as `$ARGUMENTS` (e.g., "我周围的人" or "People Around Me").

**Pipeline steps:**
1. `driving-question` — generate and validate the driving question
2. `network-map` — build the thematic network diagram
3. `inquiry-scaffold` — split into 3 progressive inquiry clues
4. `activity-design` — design activities for all 3 clues
5. `proposal-generate` — compile the complete PBL proposal

Each step pauses for your confirmation before proceeding to the next.

Use skill: "driving-question"
Then use skill: "network-map"
Then use skill: "inquiry-scaffold"
Then use skill: "activity-design"
Then use skill: "proposal-generate"
