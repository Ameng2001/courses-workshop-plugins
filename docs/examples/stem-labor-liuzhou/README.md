# STEM 劳动·柳州特色 PBL 项目案例库

面向公立幼儿园的项目式学习（PBL）课程案例库，供课题组教师参考、改编与复用。

- **课程体系**：项目式学习（PBL）· 《项目课程教学》(赫尔姆 & 凯兹) 三阶段九要素框架 + 项目五段路径（项目概览→项目标准→项目启动→项目探究→项目展示）
- **内容方向**：以幼儿劳动课为主，融合 STEM（工程/科学探究）与广西柳州本地特色
- **设计基准**：符合各年级年龄认知特点 · 公立园教学场景 · 师生比 1:10 · 低结构材料优先 · 纯中文
- **共同结构**：每份方案 = 项目导览 + 01 项目概览（含五领域覆盖表、三阶段九要素对应表）+ 主题网络图 + 02 项目标准（先前经验/4C 能力/学习目标）+ 03 项目启动（驱动性问题）+ 04 项目探究（3 条阶梯递进线索，每条 3–5 天，含活动表）+ 05 项目展示 + 教师实施提示
- **活动编码**：`PBL-C{线索号}-{两位序号}`（线索内连续、线索间从 01 重置）
- **资源四分类**：PBL 项目盒（项目专用采购）/ 探索足迹袋（幼儿个人记录与操作材料）/ 自备材料（教师班级现有）/ 多媒体（视频·歌曲·图片）

## 年龄梯度说明

- **小班**（01）重感官体验与简单照料（看摸闻尝、浇水、采摘），动作简单、时长短。
- **中班**（02·03·06）进入有规律的制作与初步社会经营（编织规律、配料摆盘、折叠成型、调查/分工），精细动作循序渐进。
- **大班**（04·05·07）承担完整工程探究闭环（观察→设计→搭建/制作→测试→改进），强调"猜想-试验-记录-改进"的迭代、几何规律与小组协作。

> 文化特色分布：03 螺蛳粉（柳州美食）、04 五菱/宝骏（柳州汽车工业）、05 柳州桥梁（桥梁博物馆）、06 紫荆花（柳州市花）、07 壮锦（广西非遗织锦）。

## 使用建议

1. **直接参考**：教师可按本园班级实际，挑选对应年级的案例直接试上或局部改编。
2. **本土替换**：广西/柳州元素（食材、桥梁、车型、蔬菜品种）可按所在地资源就近替换。
3. **材料弹性**：资源清单中"PBL 项目盒"为建议采购项，多数可用"探索足迹袋/自备材料"中的低结构、废旧材料替代。
4. **安全优先**：涉及热源、尖锐工具、攀高、绳类、食材过敏等环节，均已在各案例"教师实施提示"中标注，实施前请再次核对本班情况。
5. **周期灵活**：每条探究线索建议 3–5 天，可按幼儿兴趣与探究深度伸缩；整体控制在 3–5 周。

## 与平台流水线的关系

本案例库对应 PBL 设计流水线（`/workshop-pbl:design`）的最终产物形态（proposal）。如需在运行时工作台中迭代某个主题，可在 `.workshop/projects/{主题}/` 下新建项目，按 `驱动问题 → 网络图 → 探究线索 → 活动设计 → 预案生成` 流程展开，并以本案例库作为风格与结构参照。
