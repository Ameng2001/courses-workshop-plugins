# 区域活动：职业服饰配对

## 所属周次与关联

- **所属周次**: 第 2 周
- **关联教学活动**: 这是谁的衣服
- **投放区域**: 认知区 / 角色区

## 预期经验

1. 能根据人物或场景线索选择相应服饰。
2. 能说出“这是谁的衣服”并尝试说明理由。

## 材料投放

- 职业人物卡 8 张
- 服饰卡 16 张
- 场景底板 4 张
- 魔术贴配对板 2 套

## 指导策略

- 先从熟悉职业开始，再加入容易混淆的职业
- 鼓励幼儿用“我觉得这是……因为……”表达
- 对只会配对不会表达的幼儿，提供句式卡

## 与周计划的关联

- 可作为周一教学活动后的巩固区
- 可回收幼儿错误配对，作为周三教学活动的再讨论素材
