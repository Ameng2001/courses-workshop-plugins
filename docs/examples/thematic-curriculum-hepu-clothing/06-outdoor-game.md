# 户外游戏：小小快递员换装跑

## 所属周次与关联

- **所属周次**: 第 2 周
- **关联子主题**: 不同场景的服饰

## 预期经验

1. 知道不同任务需要选择不同服饰或装备。
2. 在跑动与取物中体验规则与合作。

## 游戏准备

- 快递箱 4 个
- 背心、帽子、手套等任务装备若干
- 起点线、终点线、障碍桶

## 游戏玩法

1. 幼儿分成两队，从起点出发。
2. 到达中途点后根据任务卡选择对应装备。
3. 穿戴后继续完成送达任务并返回。

## 场地与组织

- 使用直线往返跑道，避免交叉碰撞
- 装备放在中途补给点，方便取用

## 教师观察与安全提示

- 观察幼儿是否理解任务与装备关系
- 提醒幼儿穿戴时不推挤，返回时从外侧通道绕回
