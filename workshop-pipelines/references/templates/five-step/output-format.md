# 五步法教案输出格式

## 文档结构

五步法教案采用以下标准格式输出：

### 基本信息

| 项目 | 内容 |
|------|------|
| 活动名称 | 中文名称（英文名称） |
| 适用年龄 | 小班/中班/大班 (X-Y岁) |
| 活动领域 | 主领域 + 渗透领域 |
| 活动时长 | XX 分钟 |
| 活动类型 | 集体教学 / 小组活动 / 区角活动 |

### 教学目标

按领域分列，使用年龄适切的动词：
- 认知目标：了解/认识/知道……
- 技能目标：能够/学会/尝试……
- 情感目标：愿意/喜欢/乐于……

同时允许补充：
- **核心发展目标**：客户或园所自定义编码（如 `SE-8`）

每条目标必须：
1. 可观察、可评估
2. 与《3-6 岁儿童学习与发展指南》对齐
3. 使用该年龄段对应的动词层级

### 教学准备

分类列出：
- **环境准备**：空间布置要求
- **材料准备**：具体名称 × 数量（如"A4 白纸 25 张"）
- **经验准备**：幼儿需要的前置经验
- **多媒体准备**：PPT/视频/音频（注明来源和时长）

### 重难点

- **活动重点**：本节课最核心的学习重点
- **活动难点**：幼儿最可能出现困难的部分

### 教学过程

五步结构化表格（见 coding-spec.md 中的示例格式）：

| 环节 | 时长 | 教师行为 | 幼儿行为 | 材料 | 设计意图 | 教师观察与支持要点 |
|------|------|---------|---------|------|---------|
| S1 导入 | X min | 具体话术 | 预期反应 | 所需材料 | 教育目的 | 观察点 / 支持策略 |
| S2 探究 | X min | … | … | … | … | … |
| S3 操作 | X min | … | … | … | … | … |
| S4 总结 | X min | … | … | … | … | … |
| S5 延伸 | X min | … | … | … | … | … |

### 分层指导（可选）

| 层级 | 幼儿特征 | 指导策略 |
|------|---------|---------|
| 基础层 | 发展稍慢，需要更多支持 | 降低任务难度，增加示范 |
| 发展层 | 大部分幼儿 | 开放性指引，适度挑战 |
| 提高层 | 发展较快 | 延伸任务，鼓励创新 |

### 教学反思

留空模板，教师课后填写：
- 目标达成情况
- 幼儿表现亮点
- 需要改进的环节
- 后续跟进计划

## 格式要求

- Markdown 格式
- 表格使用标准 Markdown 语法
- 双栏版式在导出层实现；运行时文档必须保留“教师观察与支持要点”字段
- 文件名：`lesson-plan.md`
- 存储位置：`.workshop/projects/{workspace}/lesson-plan.md`
