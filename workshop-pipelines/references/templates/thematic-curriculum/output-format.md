# 主题式课程输出格式

## 月度主题交付包

标准交付包应至少包含：

1. `theme-analysis.md`
2. `month-plan.md`
3. `week-{N}-plan.md`
4. 各活动类型活动稿

## 单活动稿类型

### 教学活动

- 活动编码
- 本地目标编码体系（如适用）
- 核心发展目标
- 活动目标
- 活动准备
- 重难点
- 活动过程
- 活动延伸
- 教师观察与支持要点

### 区域活动

- 活动编码
- 本地目标编码体系（如适用）
- 预期经验
- 材料投放
- 指导策略

### 户外游戏

- 活动编码
- 本地目标编码体系（如适用）
- 预期经验
- 游戏准备
- 游戏玩法

### 生活渗透

- 活动编码
- 本地目标编码体系（如适用）
- 预期经验
- 实施要点

### 家园互动

- 活动编码
- 本地目标编码体系（如适用）
- 实施要点

## 格式适配原则

1. 平台内部先以 Markdown 结构表达语义。
2. 双栏版式视为导出层能力，不要求在 Markdown 内模拟视觉布局。
3. “教师观察与支持要点”必须作为显式字段保留。
4. 活动编码和客户本地目标编码应并列保留，不相互替代。
