# 主题式课程编码规范

## 编码格式

```text
TC-{category}-{week}-{seq}
```

示例：

- `TC-TA-W1-01`
- `TC-RA-W1-03`
- `TC-OG-W2-01`
- `TC-LR-W3-02`
- `TC-HS-W4-01`

## 活动类型缩写

| 缩写 | 类型 |
|------|------|
| `TA` | Teaching Activity / 教学活动 |
| `RA` | Region Activity / 区域活动 |
| `OG` | Outdoor Game / 户外游戏 |
| `LR` | Life Routine / 生活渗透 |
| `HS` | Home-School / 家园互动 |

## 使用规则

1. 每个活动必须唯一编码。
2. 月矩阵、周编排、活动稿应使用同一编码。
3. 客户自有发展目标编码应作为并列字段记录，不替代平台活动编码。
