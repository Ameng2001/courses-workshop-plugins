# 主题式课程活动模板

## 区域活动

- 活动名称
- 对应区域
- 对应子主题
- 活动编码
- 本地目标编码体系（如适用）
- 预期经验
- 材料投放
- 指导策略
- 观察重点

## 户外游戏

- 活动名称
- 对应子主题
- 活动编码
- 本地目标编码体系（如适用）
- 预期经验
- 游戏准备
- 游戏玩法
- 安全提示
- 观察重点

## 生活渗透

- 活动名称
- 对应子主题
- 活动编码
- 本地目标编码体系（如适用）
- 预期经验
- 实施要点
- 对应生活场景
- 观察重点

## 家园互动

- 活动名称
- 对应子主题
- 活动编码
- 本地目标编码体系（如适用）
- 实施要点
- 家长支持建议
- 回收方式 / 反馈方式

## 推荐文件命名

- 区域活动：`activities/region-{slug}.md`
- 户外游戏：`activities/outdoor-{slug}.md`
- 生活渗透：`activities/life-routine-{slug}.md`
- 家园互动：`activities/home-school-{slug}.md`

## 通用要求

1. 所有活动都应与当周子主题呼应。
2. 活动稿不要求完全复制教学活动结构。
3. 优先输出可执行的教师提示，而不是空泛理念。
4. 如客户有发展目标编码，应作为单独字段记录，不替代活动编码。
5. 活动名称要能直接回填到月矩阵和周活动安排表中。
