---
name: template-select
description: Select a teaching methodology for the workspace
skill: template-select
---

Set the active methodology template. Usage: `/workshop-templates:template-select {template-id}`

Examples:
- `/workshop-templates:template-select pbl-huamei` — 使用 PBL 华美五步法
- `/workshop-templates:template-select five-step` — 使用五步教学法
