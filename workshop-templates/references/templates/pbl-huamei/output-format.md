# PBL 华美预案输出格式

## 文档结构

PBL 预案采用华美五节结构输出：

### 01 项目概览 / Project Overview

- 主题背景（2-3 段，中英双语）
- 五大领域覆盖度表格
- 课程标准映射（3-8 条《指南》目标）

### 02 项目标准 / Project Standard

- 先备知识（认知/技能/情感三维度，每维度 5-8 条）
- 4C 技能目标（每个 C 2-3 条行为表现）
- 学习目标（6-10 条，含 4C 标签和年龄适切动词）

### 03 项目启动 / Project Launch

- 驱动性问题（含四维评分）
- 主题网络图（文本格式思维导图）
- 探究线索概览（3 条线索的关键问题和时间分配）

### 04 项目探究 / Project Inquiry

- 三条探究线索的完整活动表格
- 每个活动含：编码、名称（双语）、关键问题、内容步骤、教师指导、资源清单
- 活动编码格式：PBL-C{线索}-{序号}

### 05 项目展示 / Project Products

- 展览形式建议
- 过程档案指引
- 反思提问清单

## 格式要求

- 全文双语（中文为主，英文为辅）
- Markdown 格式
- 表格使用标准 Markdown 表格语法
- 文件名：`proposal.md`
- 存储位置：`studio/changes/{workspace}/proposal.md`
