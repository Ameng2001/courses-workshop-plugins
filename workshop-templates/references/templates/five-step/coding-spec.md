# 五步法教案编码规范 / Five-Step Lesson Coding Specification

五步法教案中各教学环节的编码规范，用于环节标识与管理。

---

## Coding Format

```
FS-S{step_number}-{sequence_number}
```

| 字段 Field | 说明 Description | 取值 Values |
|------------|-----------------|-------------|
| `FS` | 固定前缀 Five-Step (fixed prefix) | `FS` |
| `S{step_number}` | 教学环节编号 (teaching step number) | `S1`-`S5` |
| `{sequence_number}` | 环节内活动序号，两位数字 (zero-padded) | `01`, `02`, `03`... |

示例: `FS-S1-01`（导入环节第 1 个活动）, `FS-S2-02`（探究环节第 2 个活动）

---

## 环节编号映射

| 编号 | 环节名称 | 英文 | 典型时长 |
|------|---------|------|---------|
| S1 | 导入 | Introduction | 3-5 min |
| S2 | 探究 | Exploration | 8-12 min |
| S3 | 操作 | Practice | 5-8 min |
| S4 | 总结 | Summary | 3-5 min |
| S5 | 延伸 | Extension | 2-3 min |

---

## Rules

1. **环节内连续编号**：同一环节内按活动顺序编号，不跳号
2. **环节间重置**：每个新环节的序号从 `01` 重新开始
3. **一活动一编码**：每个编码对应且仅对应一个具体活动或子任务
4. **编码不可变**：编码一旦分配不重新编号

---

## 教案表格格式

每个教学环节以结构化表格呈现：

| 列 Column | 说明 Description |
|-----------|-----------------|
| **环节 Step** | 环节名称 + 时长 |
| **教师行为 Teacher Actions** | 教师的具体话术和操作 |
| **幼儿行为 Child Responses** | 预期的幼儿反应和表现 |
| **材料 Materials** | 本环节所需的具体材料 |
| **设计意图 Design Intent** | 本环节的教育目的 |

---

## 示例：认识春天的花

### 教学过程

| 环节 | 教师行为 | 幼儿行为 | 材料 | 设计意图 |
|------|---------|---------|------|---------|
| **S1 导入** (3 min) | FS-S1-01: 展示一束鲜花，问"这是什么？你在哪里见过花？" | 观察鲜花，举手分享在公园、小区见过花的经验 | 鲜花一束（含 3-4 种不同颜色的春花） | 连接已有经验，激发兴趣 |
| **S2 探究** (10 min) | FS-S2-01: 分发 4 种春花实物，引导观察"花瓣是什么形状？什么颜色？闻起来怎么样？" | 用手触摸、用鼻子闻、用眼睛看，描述花的特征 | 迎春花、桃花、樱花、郁金香实物各 5 枝；放大镜 5 个 | 多感官观察，发展观察力和语言表达 |
| **S3 操作** (7 min) | FS-S3-01: "请小朋友选一种最喜欢的春花，把它画下来" | 选择花朵，用蜡笔/水彩画出花的外形和颜色 | A4 纸 25 张；蜡笔/水彩笔每组 1 盒 | 通过绘画巩固观察经验，发展精细动作 |
| **S4 总结** (4 min) | FS-S4-01: 邀请 3 名幼儿展示画作，问"你画的是什么花？它有什么特别的地方？" | 展示作品，描述自己画的花的特征 | 展示架或黑板磁铁 | 分享交流，提升语言表达 |
| **S5 延伸** (2 min) | FS-S5-01: "明天散步时我们去幼儿园的花园找找，还有哪些花是春天开的？" | 表达期待，讨论在幼儿园见过的花 | — | 连接日常生活，延续探究兴趣 |
