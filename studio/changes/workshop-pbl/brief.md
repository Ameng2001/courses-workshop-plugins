# Plugin Brief: workshop-pbl

> Created: 2026-03-28

## Business Context

帮助课研主任从月度主题出发，系统化地完成 PBL 预案的核心创作环节：驱动性问题设计 → 主题网络图构建 → 探究线索拆分 → 活动序列编排 → 预案文档生成。这是课研流程中耗时最长（占 70%+）、门槛最高的部分。

核心用户是课研主任（张主任画像），她需要的不是"从零生成"，而是"结构化引导 + 快速迭代"——给骨架让她填肉。

## Plugin Candidates

5 个 skills 构成串行流水线：
1. driving-question — 驱动性问题生成 + 开放性验证
2. network-map — 主题网络图构建
3. inquiry-scaffold — 探究线索拆分（3 条递进）
4. activity-design — 活动序列编排（每条线索 3-5 个活动）
5. proposal-generate — 预案文档组装（华美 5 段式）

## Success Criteria

- HS-1 解决：驱动问题设计从平均 3-5 轮修改降至 1-2 轮
- HS-2 解决：活动设计时间从 70% 占比降至 40% 以下
- 新课研老师能在 1 小时内产出结构完整的预案草稿

## Notes

- 活动编码必须遵守 `PBL-C{线索号}-{序号}` 规范
- 每个活动时长约束 20-30 分钟（PreK 课时标准）
- 预案格式遵循华美 PBL 项目路径图的 5 段式结构
- 需要内置领域专家 agent 进行自动审核（课程专家、心理学家、教学设计师）
