# PBL Box Catalog / PBL 项目盒材料目录

> **说明**: 本目录以"果汁小铺"PBL 项目为样本整理，仅作为材料分类和数量的参考模板。各园所应根据自身 PBL Box 供应商的实际库存维护专属目录。
>
> **Note**: This catalog is a sample based on the juice shop PBL project. Schools should maintain their own catalog based on their PBL Box vendor's inventory.

默认班级人数：20–25 名儿童。
