## Gap Analysis

| Gap | Impact | Opportunity |
|-----|--------|-------------|
| 无课标知识库 — 每次手工查找《指南》 | 重复劳动, 遗漏风险 | 内置课标索引 + 自动匹配 |
| 无4C标准参照表 — 映射凭经验 | 映射不准确 | 内置年龄段×4C能力矩阵 |
| 无驱动问题质量标准 — 好坏靠感觉 | 反复修改, 效率低 | 开放性/可探究性自动评分 |
| 无活动时长估算模型 — 凭经验估计 | 课堂超时或不足 | 基于步骤数和年龄的时长预估 |
| 无资源数据库 — 每次重新列清单 | 遗漏, 分类错误 | PBL Box 物料数据库 + 自动匹配 |
| 无网络图生成工具 — 手绘或通用思维导图 | 格式不统一, 耗时 | 结构化网络图自动生成 |
