# Persona: 王老师（一线带班教师）

> "预案写得很好看，但到了教室里我不知道第一句话该说什么。"

┌─────────────────────────────────────────────────┐
│  王老师                                          │
│  "给我具体的提问话术和材料清单，别让我猜"         │
├─────────────────────────────────────────────────┤
│                                                 │
│  Role:        带班教师（PreK-4 班）               │
│  Age range:   23-35                             │
│  Tech level:  中                                │
│  Context:     独立带 20-25 个孩子，配 1 名保育员  │
│               每天有 30 分钟 PBL 活动时间         │
│                                                 │
│  Goals:                                         │
│  1. 快速理解预案意图，知道每个活动怎么做          │
│  2. 提前准备好所有材料，不在课上手忙脚乱          │
│  3. 灵活调整活动难度，适配班上不同发展水平的孩子   │
│                                                 │
│  Pain points:                                   │
│  1. 活动内容描述太笼统，缺乏具体话术              │
│     — severity: 高                               │
│  2. 资源清单不够具体（只写"水果"不写数量和种类）  │
│     — severity: 高                               │
│  3. 不知道如何评估孩子是否达到了学习目标           │
│     — severity: 中                               │
│                                                 │
│  Current tools:                                 │
│  - 打印的预案 PDF                                │
│  - PBL Box（统一配送的材料盒）                    │
│                                                 │
│  Success looks like:                            │
│  "预案里有分步骤的活动指引、教师提问话术示例、     │
│   完整的材料清单（含数量），我照着做就行"          │
│                                                 │
└─────────────────────────────────────────────────┘
